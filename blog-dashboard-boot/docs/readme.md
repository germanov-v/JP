



Launch app (запуск прилоежния):

```
./gradlew bootRun 
```

Запуск с параметрами:

```
./gradlew bootRun --args='--server.port=8989'
```

Запуск всех тестов:

```csharp
./gradlew test
```

Запуск юнит тестов:

```csharp
./gradlew unitTest
```

Запуск интеграционных тестов:

```csharp
./gradlew integrationTest
```

Запуск проверки https://www.baeldung.com/gradle-test-vs-check :

```bash
./gradlew check
```
